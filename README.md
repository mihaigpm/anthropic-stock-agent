# anthropic-product-sprint

# 1. Install OrbStack (Docker + K8s engine)
brew install orbstack

# 2. Install Kubernetes CLI
brew install kubectl

# 3. Install K9s (The best terminal UI for managing K8s)
brew install derailed/k9s/k9s

# 4. Install Helm (The "App Store" for Kubernetes)
brew install helm

# 5. Install HTTPie (A modern alternative to cURL for testing your API)
brew install httpie
